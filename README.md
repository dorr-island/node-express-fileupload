# Node Express File Upload
A simple file uploader using express-fileupload.

## Routes
### POST /upload
- Handles a file upload form
- Example html:
```html
<html>
  <body>
    <form ref='uploadForm' 
      id='uploadForm' 
      action='/upload' 
      method='post'
      target='_blank'
      encType="multipart/form-data">
        <input type="file" name="csvFile" />
        <input type='submit' value='Upload!' />
    </form>     
  </body>
</html>
```
